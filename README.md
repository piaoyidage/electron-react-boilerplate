
## 安装

```bash
yarn
```

**有安装问题? 参考 [debugging guide](https://github.com/electron-react-boilerplate/electron-react-boilerplate/issues/400)**

## 开始开发

`dev` 环境启动:

```bash
npm start
```

## 打包

本地平台打包:

```bash
npm run package
```
